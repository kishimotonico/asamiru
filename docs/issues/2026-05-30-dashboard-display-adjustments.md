# ダッシュボード表示調整

## 対応内容

- 気温の単位を `°` から `℃` 表記へ変更。
- 日付を `YYYY / M / D （曜）` 形式で大きく表示。
- 時刻表示に秒を追加し、将来の ON/OFF 切り替えに備えて `ClockCard` に `showSeconds` を追加。
- 運行状況を 2 列グリッドから 1 列縦並びに変更。
- 祝日表記を日付行の右側へ移動し、日付と時刻の間隔を調整。
- 気温の数字より `℃` を小さく表示。
- この先の予定の日付表記を `M/D（曜）` 形式に統一。

## 確認

- `pnpm build`
- `agent-browser` で `http://asa.localhost:1355/` を開き、表示を確認。

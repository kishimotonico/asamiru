# ダッシュボード予定カレンダー化と天気表示調整

## 対応内容

- 今日の天気カードの最高気温表示を少し小さく調整。
- 予定カードはいったん予定リスト表示へ戻し、ダッシュボードでは描画しない構成へ変更。
- 暫定的な月間カレンダー表示を `CalendarCard` として別モジュール化。
- モジュールIDに `calendar` を追加し、デフォルト有効モジュールから `schedule` を外した。
- カレンダーのヘッダー右側の重複した年月表示を削除。
- 天気カードは最高気温数字を少し小さくし、メイン天気アイコンを大きくしてバランスを調整。
- メイン天気アイコンの視覚上の偏りを補正するため、アイコン位置を微調整。
- カレンダー内の `31 TODAY` 表示を削除し、時計カードとの情報重複を解消。
- 天気カードのメイン温度表示をさらに小さくし、アイコンの高さと揃いやすく調整。
- 運行状況の正常表示を控えめなグレー系に変更し、遅延・運休時だけ目立つ配色に整理。

## 確認

- `pnpm build`
- `agent-browser` で `http://asa.localhost:1355/` を開き、デスクトップ幅とモバイル幅で表示を確認。

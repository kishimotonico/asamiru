# カレンダー予定の表示範囲拡張

## 背景

カレンダーカードの予定欄は、API 取得範囲・画面表示ともに今日と明日の2日間に固定されていた。週に数件程度の利用では予定欄が空になりやすく、先の予定を確認できない。

## 変更

- API から取得する予定範囲を、今日を含む14日間へ拡張する。
- 予定欄は日数固定ではなく、日付順の直近5件を表示する。
- 今日・明日以降の日付は `M/D（曜）` 形式で表示する。
- 複数日にまたがる終日予定は、同じ予定を日ごとに重複表示せず1件として扱う。
- デモデータへ翌週以降の予定を追加し、表示を確認できるようにする。
- 取得日数を TanStack Query のキーへ含め、範囲変更時に旧キャッシュを再利用しない。
